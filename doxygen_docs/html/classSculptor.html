Detailed Description

Author
Luis Felipe Vanin Martins

Introdução

Esse projeto consiste na criação de um escultor 3D interativo que gera figuras baseadas em cubos, criando sólidos similares a algo visto no jogo minecraft. O código desse projeto é desenvolvido inteiramenta em C++ utilizarando primariamente o estilo de programação orientado a objeto(POO).
Além da linguagem de programação C++, outras ferramentas para a vizualização das figuras geradas devem ser utilizadas, as recomendações são o geomview ou meshlab.Os softwares previamente citados foram escolhidos, pois possuem a capacidade de ler e representar arquivos do tipo .OFF que é o tipo de arquivo utilizado para se armazenar os sólidos feitos pelo programa em C++.
O tipo de arquivo .OFF é muito importante para esse projeto, pois ele é um arquivo que pode ser escrito de uma forma padronizada de maneira a ser possível a se denhar um solido 3D.

Sobre o Sculptor

Nessa classe Sculptor gerada de dentro do arquivo Sculptor.h e construido dentro do arquivo Sculptor.cpp tem a funcionalidade de criar um canvas 3D de desenho e utilizar suas funções internas(métodos) para construir objetos dentro desse canvas. A unidade de desenho utilizada por esse programa é o VOXEL que tem suas propriedade definidas no arquivo voxel.h,
Algumas das formas que podem ser geradas por essa classe são, pontos(Voxels), caixas, esferas, elipses e discos, sem falar que o programa também tem a capacidade de "cortar" os desenhos utilizando as mesmas formas sitadas como parâmetro de corte.

O método cutDisc() gera um corte cilindrico a partir de uma equação implementada no comando de iteração(for) na sua lógica. Esse método usa a função cutVoxel() e não retorna nada(void).

Parameters
xcenter: coordenada x central para o corte da base do disco/cilindro. A base depende do parâmetro "axis"
ycenter: coordenada y central para a altura do cilindro, ou seja, será o centro a ser "esticado", no caso cortado.
zcenter: coordenada z central para o corte da base do disco/cilindro. A base depende do parâmetro "axis"
r: é o raio da base do cilindro
axis: coordenada a ser extrudada(esticada) do cilindro. Esse parâmetro recebe inteiros de 0 até 2, sendo o 0 repectivo ao eixo x, o 1 ao eixo y e 2 ao z. Por padrão inicializa com o valor 1.
