\hypertarget{index_autotoc_md2}{}\doxysubsection{1. Introdução}\label{index_autotoc_md2}
Esse projeto consiste na criação de um escultor 3D interativo que gera figuras baseadas em cubos, criando sólidos similares a algo visto no jogo {\itshape minecraft}. O código desse projeto é desenvolvido inteiramenta em {\bfseries{C++}} utilizarando primariamente o estilo de programação orientado a objeto(\+POO). 

Além da linguagem de programação C++, outras ferramentas para a vizualização das figuras geradas devem ser utilizadas, as recomendações são o {\bfseries{geomview}} ou {\bfseries{meshlab}}. Os softwares previamente citados foram escolhidos, pois possuem a capacidade de ler e representar arquivos do tipo {\bfseries{.OFF}} que é o tipo de arquivo utilizado para se armazenar os sólidos feitos pelo programa em C++.

O tipo de arquivo .OFF é muito importante para esse projeto, pois ele é um arquivo que pode ser escrito de uma forma padronizada de maneira a ser possível a se denhar um solido 3D.

~\newline


O desenvolvimento desse projeto consistirá em duas etapas, sendo elas\+:

\item {\bfseries{ETAPA 1}} -\/ desenvolvimento das ferramentas de desenho 3D.
\item {\bfseries{ETAPA 2}} -\/ Criação de uma ferramenta interativa.

\item {\bfseries{src}} -\/ possui o código da lógica e implementação da classe \mbox{\hyperlink{classSculptor}{Sculptor}}.
