\hypertarget{classSculptor_intro_sec}{}\doxysubsection{Introdução}\label{classSculptor_intro_sec}
Esse projeto consiste na criação de um escultor 3D interativo que gera figuras baseadas em cubos, criando sólidos similares a algo visto no jogo minecraft. O código desse projeto é desenvolvido inteiramenta em C++ utilizarando primariamente o estilo de programação orientado a objeto(\+POO).~\newline
Além da linguagem de programação C++, outras ferramentas para a vizualização das figuras geradas devem ser utilizadas, as recomendações são o geomview ou meshlab.\+Os softwares previamente citados foram escolhidos, pois possuem a capacidade de ler e representar arquivos do tipo .OFF que é o tipo de arquivo utilizado para se armazenar os sólidos feitos pelo programa em C++.~\newline
O tipo de arquivo .OFF é muito importante para esse projeto, pois ele é um arquivo que pode ser escrito de uma forma padronizada de maneira a ser possível a se denhar um solido 3D.~\newline
\hypertarget{classSculptor_about_sec}{}\doxysubsection{Sobre o Sculptor}\label{classSculptor_about_sec}
Nessa classe \mbox{\hyperlink{classSculptor}{Sculptor}} gerada de dentro do arquivo \mbox{\hyperlink{Sculptor_8h}{Sculptor.\+h}} e construido dentro do arquivo \mbox{\hyperlink{Sculptor_8cpp}{Sculptor.\+cpp}} tem a funcionalidade de criar um canvas 3D de desenho e utilizar suas funções internas(métodos) para construir objetos dentro desse canvas. A unidade de desenho utilizada por esse programa é o VOXEL que tem suas propriedade definidas no arquivo \mbox{\hyperlink{voxel_8h}{voxel.\+h}},~\newline
Algumas das formas que podem ser geradas por essa classe são, pontos(\+Voxels), caixas, esferas, elipses e discos, sem falar que o programa também tem a capacidade de \char`\"{}cortar\char`\"{} os desenhos utilizando as mesmas formas sitadas como parâmetro de corte.
